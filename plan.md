Math Game

Setup:

- 2 players
- initalize score with 3 lives each
- random math question - verify answer

Nouns:

- player
- lives
- game output

Verbs:

- answer question
- lose a life
- announce winner
- announce score

Question Class

- generates question
- verifies if answer is correct

Player Class

- get current player
- update lives

Game Class

- set player lives
- set current player
- game loop
  - get user input
  - output score
  - check if score reaches 0
  - announce winner & loser
